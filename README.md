## Project Structure

```
nivasity/
├── admin/              # Admin dashboard and management
├── assets/            # Static assets (CSS, JS, images, vendors)
├── config/            # Configuration files
├── docs/              # Documentation
├── model/             # Backend models and data handling
├── partials/          # Reusable page components (header, footer, etc.)
├── sql/               # Database schemas
├── index.php          # Homepage
├── user.php           # User dashboard
├── support.php        # Support page
└── ...
```

## Key Features

- **Event Management**: Create and manage events
- **Ticketing System**: Issue and track tickets
- **User Dashboard**: Personalized user experience
- **Admin Panel**: Comprehensive administrative controls
- **Support System**: Integrated support ticketing
- **WhatsApp Integration**: Floating button for quick support access

## Getting Started

This is a PHP-based web application. Requirements:
- PHP 7.4 or higher
- MySQL database
- Web server (Apache/Nginx)

## Support

For support inquiries, use the WhatsApp floating button available on most pages, or visit the support page.
